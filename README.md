Quiz App using OOP
==================

A Python quiz application built with Object-Oriented Programming principles. This project demonstrates class design, data encapsulation, and interactive user experiences through a structured question-answer format.

Features
--------

Presents multiple-choice questions to users one at a time from a predefined question bank. Tracks user scores and provides immediate feedback on correct or incorrect answers. Displays final score at the end of the quiz with performance summary. Separates question data, quiz logic, and user interface into distinct classes for clean architecture.
